# Spec: Id\<T\>

## Overview

Abstract record wrapping a `Guid` for strongly-typed entity identifiers.

## API

- `Guid Value` — the underlying identifier (record positional parameter)
- `static T New()` — creates with `Guid.NewGuid()`
- `static T From(Guid value)` — creates from existing Guid, guards against Empty
- `static T Parse(string value)` — parses a string to Guid, then creates
- `implicit operator Guid(Id<T>)` — unwrap to Guid
- `explicit operator T(Guid)` — wrap Guid into Id (calls `From`)
- `ToString()` — returns `Value.ToString()`

## Constraints

- `where T : Id<T>`

## Behavior

- `New()` always produces a non-empty Guid
- `From(Guid.Empty)` throws `DomainValidationException`
- `Parse` with invalid string throws `FormatException`
- Record equality: two Ids with same Guid value are equal
- Implicit Guid conversion preserves the value
